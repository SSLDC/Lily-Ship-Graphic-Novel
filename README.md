# Lily Ship – Graphic Novel

**Lily** es una novela gráfica interactiva con temática espacial que combina narrativa ramificada, combate con naves y una presentación visual dinámica.  

El proyecto integra historia, animaciones, efectos visuales y sistemas modulares diseñados para escalar fácilmente la experiencia narrativa.

---

## Características Principales

-  Sistema de decisiones narrativas
-  Sistema modular de diálogos
-  Combate con naves y generación de enemigos
-  Control dinámico de música y video
-  Efectos visuales y transiciones animadas
-  Estructura escalable y desacoplada

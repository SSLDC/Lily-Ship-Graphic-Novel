# Lily Ship – Graphic Novel

## English

**Lily** is an interactive graphic novel with a space-themed narrative that combines branching storylines, ship combat mechanics, and dynamic visual presentation.

The project integrates story progression, animations, visual effects, and modular systems designed to scale the narrative experience efficiently.

---

## Main Features

- Narrative decision system  
- Modular dialogue system  
- Ship combat and enemy generation  
- Dynamic music and video control  
- Animated visual effects and transitions  
- Scalable and decoupled architecture
